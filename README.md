# AI Tutor — 基于 2 Sigma 理论的 AI 一对一私教系统

> **让每个人都能享受顶级私教辅导** —— 用 AI 复现 Bloom 的 "2 Sigma 奇迹"

1984 年，教育心理学家 Benjamin Bloom 发现：接受一对一辅导的学生，成绩超越了 **98%** 的普通课堂学生（高出 2 个标准差）。问题是——一对一辅导从来不具备规模化的条件。

**直到现在。**

本项目将 Claude Code 变成你的私人导师。你说出想学的主题，AI 会为你量身定制课程、逐课检验理解、根据你的薄弱点自适应调整路径——所有学习轨迹以 Markdown 文件沉淀在本地，随时回顾，永远属于你。

---

## 核心理念

| 原则 | 说明 |
|------|------|
| **掌握学习** | 只有真正理解当前内容，才推进下一课。绝不赶进度 |
| **费曼技巧** | 用最简单的语言讲清复杂概念，大量类比和真实案例 |
| **苏格拉底提问** | 用问题引导你思考，而非直接灌输答案 |
| **鼓励犯错** | 错误是诊断理解深度的最佳工具，纠偏而非批评 |

## 工作原理

```
你："我想学 X"
        │
        ▼
  ┌─────────────┐
  │ 评估知识图谱  │──→ 规划学习路径
  │ 生成第一课    │──→ 概念讲解 + 检查站问题
  └─────────────┘
        │
        ▼
  你回答检查站问题
        │
        ▼
  ┌─────────────────────────────────────┐
  │         AI 评估你的掌握程度           │
  ├──────────┬──────────┬───────────────┤
  │ 完全掌握  │ 基本掌握  │ 部分理解/未理解 │
  │ → 下一课  │→ 纠正后   │ → 换个角度重讲  │
  │          │  推进     │   生成补充课程   │
  └──────────┴──────────┴───────────────┘
        │
        ▼
  循环直到你掌握整个主题
```

## 快速开始

### 前置要求

- 安装 [Claude Code](https://claude.ai/code)（Anthropic 官方 CLI 工具）
- Git（用于版本管理你的学习记录）

### 三步上手

**1. 克隆仓库**

```bash
git clone https://github.com/chenyuantao/ai-agent-memory-learning.git
cd ai-agent-memory-learning
```

**2. 启动 Claude Code**

```bash
claude
```

Claude Code 会自动读取仓库中的 `CLAUDE.md`，加载教学系统的全部规则。

**3. 开始学习**

在对话中直接说：

```
我想学 Python 基础
```

AI 会自动：
- 创建 `Python基础/` 文件夹
- 规划完整的学习路径
- 生成第一课并等待你的回答

## 学习记录结构

每个主题自成一个文件夹，所有内容以 Markdown 存储：

```
.
├── CLAUDE.md                          ← 教学系统规则
├── AI_Agent记忆系统/                    ← 学习主题
│   ├── 进度.md                        ← 学习进度追踪
│   ├── 01_为什么Agent需要记忆.md        ← 课程 + 你的答案 + 纠偏
│   ├── 02_记忆系统的分类.md
│   ├── ...
│   └── 12_综合回顾与前沿方向.md
├── 你的下一个主题/
│   ├── 进度.md
│   └── 01_第一课.md
└── ...
```

**为什么用 Markdown？**
- 纯文本，任何编辑器都能打开
- Git 版本管理，学习轨迹永不丢失
- 数据完全在本地，属于你自己
- 搭配 Obsidian 等工具可构建个人知识库

## 自适应学习机制

这不是一个线性播放的课件，而是一个会"看你脸色"的导师：

| 你的表现 | AI 的反应 |
|----------|----------|
| 答得漂亮，能举一反三 | 直接进入下一课 |
| 大致正确，有小偏差 | 快速纠正，继续推进 |
| 概念混淆，有明显误区 | 换个角度重新解释，生成补充课程（如 `02b_补充.md`） |
| 答错或说"不知道" | 拆解问题，从更基础的地方重新来过 |

每次回答后，你的原始答案和纠偏记录都会追加到课程文件中，形成完整的学习轨迹。

## 常见问题

**Q: 可以学任何主题吗？**
A: 是的。AI 会根据你提出的主题评估知识图谱并规划路径，技术类和理论类主题都适用。

**Q: 学习记录会上传到云端吗？**
A: 不会。所有内容都是本地 Markdown 文件，你可以选择推送到自己的 GitHub 仓库做备份。

**Q: 可以中断后继续学吗？**
A: 可以。每个主题都有 `进度.md` 追踪状态，重新启动 Claude Code 后告诉它"继续学 X"即可。

**Q: 如何查看自己的学习轨迹？**
A: 直接打开对应的课程 `.md` 文件，底部有你的答案和纠偏记录；也可以查看 `进度.md` 了解整体情况。

## 背后的教育理论

本项目的教学设计基于以下经过实证检验的学习理论：

- **Bloom 的 2 Sigma 问题**（1984）—— 一对一辅导 + 掌握学习的效果远超传统课堂
- **费曼学习法** —— 用简单语言解释复杂概念，检验是否真正理解
- **苏格拉底式教学** —— 通过提问引导学生自主发现答案
- **脚手架理论**（Vygotsky）—— 从已知出发，逐步搭建通往未知的桥梁
- **间隔重复与检索练习** —— 通过检查站问题强化长期记忆
